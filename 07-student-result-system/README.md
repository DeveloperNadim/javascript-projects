# 🎓 Student Result System

A simple JavaScript project that calculates a student's total marks, average marks, and grade using **Objects**, **Arrays**, **Loops**, and **Switch Case**.

---

## 📌 Features

- Store student information using an object
- Calculate total marks using a `for...of` loop
- Calculate average marks
- Assign grades using `switch(true)`
- Display the student's result in the terminal

---

## 🛠️ Technologies Used

- JavaScript (ES6)
- Node.js (for running in the terminal)

## 📊 Grade System

| Average Marks | Grade |
|--------------:|:-----:|
| 80 - 100 | A+ |
| 70 - 79 | A |
| 60 - 69 | A- |
| 50 - 59 | B |
| Below 50 | Fail |

1. Clone the repository

```bash

```

2. Open the project folder

```bash

```

3. Run the program

