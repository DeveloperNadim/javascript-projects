# 🧮 Simple Calculator

A beginner-friendly calculator built with **JavaScript** and **Node.js**. This project performs basic arithmetic operations based on the operator entered by the user.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- User input using Node.js `readline`
- Operator selection using `switch` statement

## 📌 Current Limitations

- Numbers are hardcoded (`10` and `20`).
- Invalid operators are not handled.
- Division by zero is not checked.
